# AI-Companion
A local AI desktop assistant powered by Whisper, LLM and Vision.
AI Companion is a local AI desktop assistant that can:

🎬 Real-time movie subtitles

🎮 Game voice translation

👨‍🏫 Meeting transcription

📝 AI meeting summary

🌏 Multi-language translation

🤖 Local LLM

Powered by

Whisper

PyTorch

Ollama

Qwen

PySide6
