1. Project Vision（项目愿景）
  AI Companion 是什么？
  AI Companion 是一个本地 AI 桌面助手，它能够倾听、理解并协助用户完成任务。

2. Why（为什么做）
   日本 Prime 没中文字幕。
  英文 YouTube 听不懂。
  日本老师开会来不及记。
  PUBG 外国队友交流困难。
  线上课程需要实时翻译。
  希望所有数据都能本地处理，保护隐私。

3. Target Users（目标用户）
  留学生
  开发者
  AI 研究人员
  游戏玩家
  经常参加国际会议的人

4. MVP（第一版必须完成）
   V0.1 Alpha
      ✅ 系统声音采集
      ✅ Whisper GPU 实时识别
      ✅ 实时字幕
      ✅ 中英/英中翻译
      ✅ 悬浮字幕窗口

5. Long-term Goals（长期目标）
  Meeting Mode
  Movie Mode
  Game Mode
  Study Mode
  Vision
  Memory
  Agent
  Plugin System

6.Project Principles（项目原则）
  - Local First（默认本地运行）
  - Privacy First（默认保护用户隐私）
  - Modular Design（模块化设计）
  - Plugin Friendly（插件化扩展）
  - Cross Platform（跨平台支持）

7. Future Vision（未来愿景）
   AI Companion aims to become an intelligent desktop operating companion that listens, understands, remembers and assists users in daily life.
